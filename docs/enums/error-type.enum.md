## Error Type

An object that defines the type of exception or error.

## Parameters

| Values   | Description       |
|:---------|:------------------|
| `Client` | Client error type |
| `Server` | Server error type |

## Example

  ```javascript
import { ErrorType } from 'custom-error-service';

const type = ErrorType.Client;
```
